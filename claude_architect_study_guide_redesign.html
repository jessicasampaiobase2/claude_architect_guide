27% Agentic Architecture & Orchestration

O que estudar: Ciclo de vida do agentic loop (stop_reason "tool_use" vs "end_turn"), padrões coordinator-subagent, hook interception, session state management, task decomposition.

Conceitos-chave: Como o modelo decide qual tool chamar a cada iteração. Como passar contexto entre subagentes. Evitar anti-patterns (loop caps arbitrários, parsing de linguagem natural para terminação). Instâncias independentes de review são mais eficazes que self-review.

Recurso principal: Curso "Introduction to Subagents" no Skilljar + documentação do Agent SDK.

20% Claude Code Configuration & Workflows

O que estudar: Hierarquia de CLAUDE.md files, slash commands customizados, Agent Skills com SKILL.md frontmatter, plan mode e quando usá-lo, integração com CI/CD pipelines, Message Batches API para processamento em volume.

Conceitos-chave: Como configurar Claude Code para times. Diferença entre plan mode e execução direta. Hooks e permissões. Gestão de contexto em sessões longas.

Recurso principal: "Claude Code in Action" + "Introduction to Agent Skills" no Skilljar.

20% Prompt Engineering & Structured Output

O que estudar: System prompt engineering (princípios vs condicionais, dilution, few-shot examples), JSON schema enforcement, Pydantic para validação, chain-of-thought, validation loops para saídas críticas.

Conceitos-chave: Quando usar few-shot vs zero-shot. Como estruturar system prompts sem diluição. Enforcement de schema. Reduzir false positives em pipelines de classificação.

Recurso principal: "Building with the Claude API" + docs de structured outputs.

18% Tool Design & MCP Integration

O que estudar: MCP primitives (tools, resources, prompts), trust model, configuração de transporte (SSE, stdio), tool description best practices, structured error responses, distribuição de ferramentas entre agents, MCP Advanced Topics.

Conceitos-chave: Como escrever tool descriptions que o modelo interpreta corretamente. Quando um tool deve retornar erro vs tentar resolver. Sampling, notifications, capability negotiation.

Recurso principal: "Introduction to MCP" e "MCP: Advanced Topics" no Skilljar.

15% Context Management & Reliability

O que estudar: Context window limits, estratégias de prompt caching, RAG patterns, como estruturar informação em contextos longos, escalation paths, graceful degradation, handoff design para customer support.

Conceitos-chave: Quando usar cache vs não. Como evitar "attention dilution" em revisões de arquivos grandes. Retries vs fail. Quando escalar para humano vs resolver autonomamente.

Recurso principal: "AI Capabilities and Limitations" + docs de prompt caching.
